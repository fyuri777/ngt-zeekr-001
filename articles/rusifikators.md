## Обзор рынка русификации

На Zeekr 001 сложился рынок из пяти основных поставщиков русификации, каждый со своей специализацией и ценовой политикой. Выбор зависит от бюджета, требований к полноте перевода (планшет, приборка, задний экран) и готовности отдать контроль над инженерным меню.

**Infohub (zeekr_infohub)** — бесплатная базовая русификация при открытой инженерке, расширенная версия ~10 тыс. руб. Установка удалённо через утилиту zeekrtool или ADB Toolkit за 5–10 минут. Русификация планшета + приборки (последние версии) + галереи + сценариев. Пожизненные бесплатные обновления, не закрывает ADB. Быстрее всех выпускает пакеты под новые прошивки — иногда раньше официального OTA. 👥 CONSENSUS

**GMC** — 50 тыс. руб. на весь срок владения, бесплатное восстановление после любых обновлений. Включает голосовое управление с автопаузой музыки, проекцию навигации на лобовое стекло (HUD), приложение для прогрева и мониторинга батареи, разделение экрана, кастомные звуки. Не закрывает ADB. Требуется физический визит для установки.

**Xanavi (Ксанави)** — 80–130 тыс. руб. за полный пакет (планшет + приборка + задний экран). Обновление прошивки по кабелю — 60 тыс. руб. Бесплатное восстановление русификации для первого владельца. Голосовое управление на русском, разделение экрана на 3 части. Закрывает инженерное меню и ADB после установки. Москва и регионы, физические сервисы.

**XMETA** — 40 тыс. руб. (ADB открыт), +25 тыс. если ADB закрыт. Подписка: 10 тыс./год или 20 тыс. навсегда (lifetime). Бесплатны только минорные обновления (6.1.2→6.1.3), мажорные (6.1→6.2) — платно. Специализация на телематике и связи, Telegram-бот для push-обновлений. Москва, Питер, удалённая работа.

**ZeeAppStore / Rustor** — не полная русификация, а магазин адаптированных приложений (Яндекс.Карты, Яндекс.Музыка с крупными кнопками для планшета). Платная подписка. Меняет подпись разработчика — конфликт с оригинальными APK Яндекса.

## Обновление прошивки и русификация: как не сломать систему

👥 CONSENSUS

Обновление по воздуху (OTA) с установленной русификацией — почти гарантированная поломка. Русификация ставится поверх заводской прошивки, при OTA системные файлы перезаписываются — лаунчер слетает, локаль переключается на китайский, появляются баги: шторка не открывается, сиденья не регулируются, климат глючит. По данным сообщества, проблемы возникают в 99% случаев.

**Безопасный алгоритм обновления:**

1. Переключить локаль на китайскую (через «лягушку» или приложение смены языка).
2. Перезагрузить планшет.
3. Обновить по кабелю **USB A–USB A** через сервис или удалённо с помощью русификатора. Кабель **USB A–USB C** — для обновления прошивки через компьютер.
4. Восстановить русификацию у своего поставщика.

GMC и Infohub восстанавливают бесплатно на весь срок владения. Xanavi — бесплатно для первого владельца. XMETA — только минорные обновления бесплатно.

Штатный китайский язык + английская локаль обновляются по воздуху без проблем. Для дореста существует инструкция OTA-обновления через DNS Changer + ADB, но проще и безопаснее — через компьютер по кабелю.

## Инженерное меню и ADB: контроль над машиной

🔥 DEBATED

Доступ к инженерному меню (ADB) — ключевой параметр при выборе русификатора. Через ADB устанавливаются приложения, меняется лаунчер, открываются скрытые функции, чистится кэш при сбоях. Без ADB владелец зависит от русификатора для любого действия.

**Кто закрывает:** Xanavi, Provolta, Fenzy — ставят динамический пароль на инженерное меню и блокируют ADB.

**Кто оставляет открытым:** GMC, Infohub — намеренно не блокируют доступ.

Обходной путь — приложение **zSupport** из ZeeAppStore/RuStore: переключает режим USB-порта и даёт доступ к ADB без инженерки. Но это костыль, не полноценная замена.

Стоимость открытия инженерки зависит от версии прошивки: **5–10 тыс. руб.** на версии 6.0.8, **50–80 тыс. руб.** на версии 6.1.2+. С версии 6.1.2 завод ввёл пароль на ADB — разница в цене в 10 раз. Если планируете самостоятельную русификацию через Infohub — открывать инженерку выгоднее до обновления на 6.1.2.

## Перевод приборной панели

Приборка и камеры работают на QNX (не Android), поэтому их русификация технически сложнее планшета. Не все поставщики умеют это делать.

Xanavi, GMC, XMETA и Infohub (последние версии) — переводят и планшет, и приборку. Бюджетные решения — только планшет, приборка остаётся на английском или китайском. Отдельная русификация приборки — **20–25 тыс. руб.**

На рестайлингах (6.7+) у продвинутых поставщиков есть решения без разборки и снятия блоков. На дорестах требовалась физическая работа с блоком. Завод добавляет зачатки английской локализации в приборку начиная с версии 6.1+.

⚠️ STALE — информация по приборке быстро устаревает: поставщики регулярно расширяют поддержку новых прошивок. Уточняйте актуальность для вашей версии ОС.

## Альтернатива: штатный английский без русификации

На прошивке 6.x для рестайлинга доступна штатная английская локаль. На версии 6.5+ через ZSupport можно включить встроенный русский, покрывающий ~95% интерфейса (кроме приборки). Минимальный набор: штатный русский через Z-Support (переключение «лягушкой»), русские пакеты настроек и климата, лаунчер с Яндекс.Картами. Приборка остаётся китайской.

Многие владельцы год и более ездят на штатном английском без русификации — обновления приходят по воздуху без проблем, нет расходов на восстановление после каждого обновления. Один раз настроить через переводчик и забыть.

## Устранение проблем после обновления

Если после обновления или OTA слетел лаунчер или русификация работает криво — стандартные действия через ADB:

- **Слетевший лаунчер:** `adb uninstall com.zeekr.carlauncher3d` (или `ecarx.launcher3`), затем `adb shell pm clear com.zeekr.carlauncher3d` — установить актуальную версию под текущую прошивку.
- **Нерабочая шторка (SystemUI):** почистить кэш через ADB, перезагрузить.
- **Конфликты приложений:** старые APK из data/ конфликтуют с новой прошивкой — удалить через ADB, восстановить русификацию заново.

Частая ошибка — лаунчер или приложение климата от другой версии ОС. Лаунчер от 6.0.8 не работает на 6.1.2 и наоборот. Решение: обновить ОС до последней, затем обновить системные приложения под эту версию.

Если нет доступа к ADB — обращайтесь к автору русификации для переустановки. У GMC и Infohub восстановление занимает 30–45 минут, часто удалённо.

## Спорные моменты

### Русификация vs штатный английский — нужна ли вообще?

🔥 ОБСУЖДАЕТСЯ

**Лагерь A:** Лучше ездить на штатном английском — стабильнее, обновления по воздуху без проблем, не нужно платить за восстановление. На 6.5+ встроенный русский через ZSupport покрывает 95% интерфейса. Английский с завода появился, разово настроить через переводчик и забыть. Экономия десятков тысяч рублей и нервов.

**Лагерь B:** Русификация нужна для полноценного использования — перевод всех меню включая приборку, голосовое управление с автопаузой музыки, навигация на HUD-проекции, разделение экрана, кастомные звуки, сценарии. Встроенный русский неполный — приборка китайская, нет голосового ассистента. Современные русификации (Infohub, GMC) стабильны и быстро обновляются.

### Infohub (бесплатная) vs платные русификации

🔥 ОБСУЖДАЕТСЯ

**Лагерь A:** Infohub — лучший выбор по соотношению цена/качество. Бесплатная, быстрые обновления, открытая инженерка, пожизненная поддержка. Экономия 60–80 тыс. руб. по сравнению с Xanavi. Не перекомпилирует APK — минимальные риски глюков.

**Лагерь B:** Платные русификации оправданы. GMC и Xanavi предлагают функции, которых нет у Infohub: голосовое управление, проекция навигации на лобовое стекло, разделение экрана на 3 части, фирменное приложение для прогрева. Качество техподдержки и гарантия стабильности стоят своих денег.

### Закрытие инженерного меню — защита бизнеса или lock-in?

🔥 ОБСУЖДАЕТСЯ

**Лагерь A:** Закрытие инженерки — недобросовестная практика. Привязывает владельца к одному мастеру, создаёт риски при необходимости самостоятельно исправить проблемы, ограничивает права собственника. При смене русификатора — factory reset или плата за разблокировку.

**Лагерь B:** Русификаторы защищают свой труд и интеллектуальную собственность. Без защиты от копирования бизнес-модель нежизнеспособна. Пароль на ADB с версии 6.1.2 — заводской, не ставится русификаторами.

### Обновлять прошивку или оставаться на стабильной?

🔥 ОБСУЖДАЕТСЯ

**Лагерь A:** Стабильная версия (5.2.x для дореста, 6.0.8/6.5.2 для реста) уже отлажена. Обновления приносят слёт русификации, новые глюки, рост расхода энергии. Владельцы на прошивке 1.0 ездят 2 зимы и 50 тыс. км без проблем. 90% обращений — проблемы с языком после обновлений.

**Лагерь B:** Новые версии исправляют критические баги: отказ тормозов на 6.0.7, проблемы с пневмой. Добавляют функции: камеры в реальном времени, режим парковки, лидар видит светофоры, пневма, улучшенный звук. Критические обновления безопасности важнее удобства русификации.

## Предупреждения

- ⚠️ **OTA с русификацией = поломка.** Обновление по воздуху с установленной русификацией почти гарантированно ломает систему. Обновляйтесь только по кабелю через сервис или удалённо с русификатором.
- ⚠️ **Новый владелец платит заново.** Большинство поставщиков не переносят гарантию: восстановление для второго владельца — 80–120 тыс. руб. Только GMC и Infohub восстанавливают на весь срок владения машины.
- ⚠️ **Закрытый ADB = зависимость.** Xanavi, Provolta, Fenzy закрывают инженерное меню. Сменить русификатора можно только через factory reset (потеря всех данных) или платную разблокировку.
- ⚠️ **Приборка не у всех.** Не все русификации переводят приборную панель. Доплата — 20–25 тыс. руб. Уточняйте до установки.
- ⚠️ **Несовместимая русификация блокирует функции.** Кривая версия может сломать: бардачок (не открывается кнопкой), климат, задний экран, режим отдыха, запись регистратора. Требуйте проверку всех функций после установки.
- ⚠️ **ZeeAppStore конфликтует с оригинальными APK.** Приложения из ZeeAppStore нельзя ставить одновременно с оригинальными Яндекс-приложениями — разные подписи. Удалите все оригинальные версии перед установкой, иначе в лаунчере пропадёт навигация.
- ⚠️ **Factory reset ≠ Reset to default.** Сброс к заводским настройкам (factory reset) удаляет русификацию, все приложения и данные безвозвратно. Reset to default сбрасывает только пользовательские настройки (память сидений, расписание подогрева), русификация сохраняется. Путать нельзя.
- ⚠️ **Связь на 6.7+.** Массовые проблемы с переходом 4G→E на прошивке 6.7.x — не связаны с русификацией, но проявляются после обновления.
